# web3 app

## 环境要求
node版本-20+

## 安装依赖
npm install

## 启动项目
npm start
## 功能
- 1. 连接以太坊网络
- 2. 获取账户余额
- 3. 获取交易记录
- 4. 发送交易
- 5. 获取区块信息
- 6. 获取合约信息
- 7. 调用合约方法
- 8. 部署合约
- 9. 获取合约事件
